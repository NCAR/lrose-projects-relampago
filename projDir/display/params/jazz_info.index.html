JAZZ DISPLAY FOR PECAN

Jazz is a Java application launched via Java Web-Start (javaws)

You need the Java JRE

You need to have the latest version of the Java Runtime Environment (JRE) from Oracle, including the javaws application launcher.

You can download Java from here, or from the Oracle site

WINDOWS and LINUX

On WINDOWS and LINUX, you can launch Jazz from within your browser, by clicking on

   the Jazz JNLP file.

You will need to set your browser preferences so open a .jnlp file using javaws.

NOTE: you can also run javaws from the command line in a terminal. Just run the following:

   javaws http://ral.ucar.edu/projects/pecan/apps/jazz/jazz.jnlp

MAC

On a MAC, if you are using Safari, javaws may be disabled by default

Instead, open a terminal and run the following command:

   javaws http://ral.ucar.edu/projects/pecan/apps/jazz/jazz.jnlp

Alternatively you can install Chrome and use that to launch Jazz in the same way as for WINDOWS and LINUX.
